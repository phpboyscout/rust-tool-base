# Rust Tool Base (RTB)

RTB is a Rust sibling of [Go Tool Base (GTB)](https://gtb.phpboyscout.uk). It
is a batteries-included application framework for Rust CLI tools, not a port
of GTB's Go API — it embraces idiomatic Rust from top to bottom.

- **[About / Why RTB?](about/why-rtb.md)** — philosophy and scope.
- **[Ecosystem Survey](about/ecosystem-survey.md)** — the crates RTB wraps
  and why.
- **[Authoritative Spec](development/specs/rust-tool-base.md)** — the single
  source of truth for every subsystem.
- **[Concepts](concepts/index.md)** — architecture, App context, config,
  assets, error handling.
- **[Components](components/index.md)** — per-crate API contracts.
